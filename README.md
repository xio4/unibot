Unibot
==============

Unibot is universal bot for lineage-like games (Archeage, Lineage2 etc). It supports scripts on pure java and plugins for encrypt and decrypt packets.

Installation
============

git clone https://github.com/xio4/unibot

Compilation
===========

mvn package clean
